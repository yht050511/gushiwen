# 诗词数据库(古诗文网)

## 介绍

共十万首诗(108197)的中华诗词数据库.由对[古诗文网](https://www.gushiwen.cn/)进行爬取得到.

+ 爬取时间: 2022-12-28

+ 字段:
  
  + id: 诗词的编号
  + href: 古诗文网对应链接,示例:`/shiwenv_161d06b0b556.aspx`
  + title: 诗词标题
  + author: 作者
  + dynasty: 朝代
  + content: 诗词内容
  + sons: 各种译文/赏析/背景之类
  + links: 相关的诗词链接(猜您喜欢)

+ 文件: 
  
  + `gushiwen.sql.gz`
  
  + `gushiwen.json.gz`
